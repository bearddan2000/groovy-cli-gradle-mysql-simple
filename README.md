# groovy-cli-gradle-mysql-simple

## Description
A groovy gradle build, that connects to mysql database.

## Tech stack
- groovy
- gradle
  - 6.8.2

## Docker stack
- mariadb:latest
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
